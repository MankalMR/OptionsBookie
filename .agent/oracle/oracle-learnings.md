## 2024-03-19 - Understanding Current Capabilities
**Learning:** OptionsBookie already has chained transactions (rolled trades) visually grouped together to track campaigns (Idea 2). It also already displays Capital Efficiency / Return on Capital (RoC) and other critical metrics in the History & Analytics tab (Idea 3).
**Action:** Avoid proposing these features in the future. Focus on new functionality that extends these existing areas or addresses unmet needs.
