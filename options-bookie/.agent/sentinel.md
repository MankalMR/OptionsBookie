## 2026-03-09 - Missing Auth Checks in API Routes | **Learning:** External API wrapper routes like `/api/stock-prices` lacked `getServerSession` checks, potentially allowing unauthenticated users to use our application as a free proxy to external APIs like Alpha Vantage, risking rate limit exhaustion or unexpected costs. | **Prevention:** Always enforce `getServerSession` checks on all non-demo `/api/*` routes, even those that do not directly interact with user-specific database data.
